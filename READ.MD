# Project Name

A online-trending platform full-stack MERN application Built using MongoDB, Express.js, React.js, and Node.js

## Feature
-User Authentication , and Authorization
-Responsive User Interfase 
-Rest Api integration
-Admin DashBoard with to Buy and sell order
-To manage to user Balance
-To store all the activity in the tranction
-Secure Database Integration

## Tech stack
